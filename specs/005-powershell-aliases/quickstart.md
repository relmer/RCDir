# Quickstart: PowerShell Alias Configuration

**Feature**: 005-powershell-aliases | **Date**: 2026-03-25

## Module Overview

| Module | File(s) | Responsibility |
|--------|---------|---------------|
| **ProfilePathResolver** | `profile_path_resolver.rs` | Detect parent PS version, resolve 4 profile paths for that version |
| **ProfileFileManager** | `profile_file_manager.rs` | Read/write/backup profile files, parse/find/replace marker blocks |
| **AliasBlockGenerator** | `alias_block_generator.rs` | Generate PowerShell function code from AliasConfig |
| **TuiWidgets** | `tui_widgets.rs` | Text input, radio buttons, checkboxes, confirmation — raw console input |
| **AliasManager** | `alias_manager.rs` | Orchestrate the three flows: set, get, remove |

## Build & Test

```powershell
# Build and test:
cargo build
cargo test

# Or use the existing VS Code / PowerShell scripts:
pwsh -File scripts/Build.ps1
pwsh -File scripts/RunTests.ps1
```

## Key Integration Points

### CommandLine: Adding new switches

In `command_line.rs`, add fields to the `CommandLine` struct:
```rust
pub set_aliases:    bool,
pub get_aliases:    bool,
pub remove_aliases: bool,
pub what_if:        bool,
```

In the switch parsing logic, add entries for `set-aliases`, `get-aliases`, `remove-aliases`, `whatif`.

In switch validation, enforce:
- `--set-aliases`, `--get-aliases`, `--remove-aliases` mutually exclusive with each other and with listing switches
- `--whatif` only valid alongside `--set-aliases` or `--remove-aliases`

### main.rs: Dispatch

Early in the command processing, after switch parsing, before directory listing:
```rust
if cl.set_aliases || cl.get_aliases || cl.remove_aliases {
    alias_manager::run (&cl)?;
    return Ok (());
}
```

```text
################################################################################

function dt  { d --tree @args  }
function dw  { d -w @args      }
function dd  { d -a:d @args    }
function ds  { d -s @args      }
function dsb { d -s -b @args   }

################################################################################
#  End RCDir Aliases
################################################################################
```
